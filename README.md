# Rule-based-chatbot
A simple chatbot that responds to user messages using predefined logic.. The chatbot processes user inputs and returns responses based on programmed conversation rules.
## Features
- Interactive chatbot interface
- Rule-based responses
- FAQ section
- Health plan information
- Contact details and location support
- Responsive design for desktops and mobile

## Technologies Used
- HTML
- CSS
- Javascript

## How to Run
- Download or clone the respository
- Open 'index.html' on your browser

##Project Purpose
- The goal of this project was to practice frontend development, chatbot logic, user interaction handeling, and structuring a simple support chatbot for a healthcare organization.
